As previsões usam dados da HLTV. Como a HLTV bloqueia requisições de navegador (Cloudflare), os dados são coletados por um script Node que gera este index.html já com os resultados. Rode localmente:

npm install
node build.mjs

Depois é só abrir o index.html gerado. Sem chave de API.

Opções

• NUM_GAMES=8 node build.mjs — nº de jogos a prever.
• DELAY_MS=3000 node build.mjs — pausa entre requisições (anti-ban da HLTV).
• FIRST_MODE=seed|underdog|coin node build.mjs — quem escolhe o 1º mapa no veto.
