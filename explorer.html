Busca e mostra o JSON cru de cada endpoint que o preditor usa, lista os campos detectados e verifica se o que o modelo precisa existe. Respeita o limite de 5 req/min (fila de ~13s) e usa cache de 6h. Clique nos chips para puxar dados encadeados (uma partida → seus mapas; um time → seu map pool).
